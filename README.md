# Event Equipment Tracker op GitHub

## Stap 1: Repository aanmaken

1. Ga naar [github.com/new](https://github.com/new)
2. Vul in:
   - **Repository name**: `event-equipment-tracker`
   - **Description**: "Equipment checklist voor live events"
   - **Public** (aangevinkt)
3. Klik **Create repository**

## Stap 2: Bestanden uploaden

1. Klik in je repo op **Add file** → **Upload files**
2. Sleep de volgende bestanden naar de upload zone:
   - `index.html` (de app zelf)
   - `README.md` (deze instructies)

3. Commit met bericht: "Initial commit"

## Stap 3: GitHub Pages aanzetten

1. Ga naar **Settings** (rechts in je repo)
2. Scroll naar **Pages** (linkermenu)
3. Onder "Source" kies **Deploy from a branch**
4. Branch: `main` → folder: `/root`
5. Klik **Save**

GitHub zal nu de app hosten op: `https://jouwusernaam.github.io/event-equipment-tracker/`

## Stap 4: Klaar!

Open die link in je browser en de app werkt! ✅

---

## Wat je nodig hebt

- GitHub account (gratis op github.com)
- Die `index.html` file

## Hoe werkt het?

De app:
- Draait 100% in je browser
- Geen server nodig
- Gebruikt gratis Claude API voor analyse (zelf API-key niet nodig — je geeft die in de app)
- Gegevens worden lokaal opgeslagen (niet naar de cloud)

## Updates maken

Wil je items toevoegen? Edit `index.html` en commit naar GitHub — de website update automatisch!
